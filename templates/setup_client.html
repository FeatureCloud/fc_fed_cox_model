    How to Run the
    Federated Cox PH App

    To run the Cox model, the data needs to be uploaded. The master has already set some parameter to run the Cox regression.

    Back to Setup Back to Criteria

    Upload File

    Select the file of your desired dataset.

    File Type

    You can either upload a CSV file or a TSV file.

    Duration and Event Column

    The name of the duration and event column needs to be the same for every client.

    Maximum Number of Iterations and Convergence Criteria

    Cox model is performed with max_steps and precision criteria set by the master.

    High-Dimensional-Data

    Lasso, ridge or elastic net penalized regression are available for the federated Cox model. The penalization term defines the regularization strength. When an elastic net regression is performed, an additional parameter l1_ratio is needed. L1_ratio controls the weighting between the L1 & L2 penalization term.
